# usecase

## MVP

### 알림 서비스 등록

- servicename 을 설정하고 생성.
- 생성 시 `API Key`, 알림을 보낼 수 있는 api의 endpoint가 제공

**api key**

- 키는 재생성이 가능하다.

### 채널 공유

- 다른 유저가 만든 채널을 링크로 공유하는 기능
- 링크로 채널에 조인하는 경우 해당 채널에 대해 같은 알림을 받음

### 액션 알림

> 액션 알림은 accept,reject 같은 상호작용을 할 수 있는 알림을 말함

- 알림 클릭 시 화면에서 해당 알림에 대해 수락,거절 을 할 수 있음
- 액션 후 상태가 변경되면 api 호출한 서버쪽에서 상태값을 pull방식으로 받을 수 있음
